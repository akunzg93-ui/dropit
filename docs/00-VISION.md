# Dropit

> Versión del documento: 1.0
>
> Estado: Oficial

---

# Visión

Dropit es una plataforma tecnológica que conecta vendedores, clientes y establecimientos para realizar entregas seguras utilizando una red de puntos de entrega.

A diferencia de una paquetería tradicional, Dropit no busca transportar paquetes entre ciudades mediante centros de distribución propios. Su objetivo es aprovechar establecimientos existentes para crear una red de entrega más flexible, económica y accesible.

La plataforma coordina el flujo completo del pedido, valida cada etapa mediante códigos de seguridad y proporciona trazabilidad desde la creación hasta la entrega.

---

# Problema que resuelve

Actualmente existen miles de ventas entre personas y pequeños comercios donde la entrega representa el mayor problema.

Las alternativas suelen ser:

- Entregas personales.
- Acuerdos por WhatsApp.
- Encontrarse en estaciones de metro o plazas comerciales.
- Servicios de mensajería costosos.
- Falta de confianza entre comprador y vendedor.

Este mercado ya existe, pero opera de forma informal.

Dropit busca formalizar ese comportamiento mediante una plataforma segura.

---

# Propuesta de valor

Dropit permite que un vendedor entregue un producto en un establecimiento participante para que posteriormente el cliente pueda recogerlo cuando le resulte conveniente.

La plataforma ofrece:

- Seguridad mediante códigos de validación.
- Seguimiento del pedido.
- Flexibilidad de horarios.
- Red de establecimientos participantes.
- Costos accesibles.
- Experiencia sencilla para todas las partes.

---

# Qué NO es Dropit

Dropit no pretende:

- Reemplazar a las grandes paqueterías nacionales.
- Competir en envíos de larga distancia.
- Operar centros de distribución propios.
- Realizar transporte de última milla con repartidores propios.

El enfoque inicial es resolver entregas locales utilizando establecimientos existentes.

---

# Usuarios del sistema

Actualmente existen cuatro tipos de usuarios.

## Cliente

Persona que recibirá el producto.

## Vendedor

Persona o negocio que genera el pedido.

## Establecimiento

Negocio autorizado para recibir y entregar paquetes.

## Administrador

Usuario responsable de la operación de la plataforma.

---

# Filosofía del producto

Dropit parte de una idea sencilla:

> La plataforma no inventa un nuevo comportamiento.
>
> Formaliza un comportamiento que ya existe.

Miles de personas ya realizan entregas en cafeterías, tiendas, estaciones de transporte y plazas comerciales.

Dropit convierte ese proceso informal en un flujo seguro, trazable y escalable.

---

# Objetivo

Construir la red de puntos de entrega más grande de México para facilitar el comercio entre personas y pequeños negocios.

---

# Principios

Todo cambio dentro del producto debe cumplir con los siguientes objetivos:

- Simplicidad.
- Seguridad.
- Escalabilidad.
- Trazabilidad.
- Buena experiencia de usuario.

Cuando exista un conflicto entre agregar funcionalidades o mantener un flujo simple, siempre se priorizará la simplicidad.
